### Ragflow是什么？

[RAGFlow](https://ragflow.io/) 是一款基于深度文档理解构建的开源 RAG（Retrieval-Augmented Generation）引擎。为帮助理解源码，开放详细设计文档，使开发者更容易上手与阅读。
